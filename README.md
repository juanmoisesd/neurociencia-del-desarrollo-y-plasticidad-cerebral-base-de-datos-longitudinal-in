## Overview
Base de datos longitudinal sobre neurociencia del desarrollo y plasticidad cerebral (0-32 a&ntilde;os). Contiene 13 modulos: (1) volumen cerebral por edad; (2) grosor cortical por region en mm; (3) poda sinaptica por edad; (4) periodos criticos de plasticidad con onset y offset; (5) efectos ambientales cuantificados: estres, desnutricion, enriquecimiento, toxinas; (6) estudios longitudinales clave
